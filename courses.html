Ruby on Rails 5 na Prática (Portuguese)

Aprenda a construir aplicações Ruby on Rails na prática

Quer aprender a criar Aplicações Web com agilidade, utilizando um dos frameworks mais modernos? Este é o curso que você precisa para criar todos os fundamentos necessários para desenvolver Aplicações Web dinâmicas, responsivas e utilizando banco de dados.

Neste curso iremos construir uma aplicação completa, desde a concepção dos modelos em banco dados até o cadastro e autenticação de usuário.

Aqui veremos os princípios do MVC, Validações de formulários, relacionamento entre modelos e o uso do Twitter Bootstrap para criação de frontend resposivos.

No final deste curso, você terá uma aplicação completa, pronta para ser colocada online.
